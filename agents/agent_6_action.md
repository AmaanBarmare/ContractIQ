# Agent 6: Action & Generation Agent

**Single job:** Convert the approved decision package into every artifact the team needs to act. Nothing leaves without a human approval stamp.

---

## Responsibilities

1. Receive the decision package from Watsonx Orchestrate (after human approval gate is passed)
2. Generate all relevant artifacts for the recommendation type
3. Mark every artifact as `DRAFT — PENDING APPROVAL`
4. Present artifacts to the user in the Artifact Approval UI
5. Wait for user review — do not proceed until explicit approval is given
6. On approval: log to Redis audit stream, mark workflow complete, update Renewal Command Center
7. On rejection or edit: return revised artifact to user for re-approval

## Approval Gate Implementation

```python
async def generate_and_gate_artifacts(decision_package: dict, workflow_id: str):
    # Generate all artifacts
    artifacts = await generate_artifacts(decision_package)
    
    # Mark all as pending approval
    for artifact in artifacts:
        artifact["status"] = "DRAFT_PENDING_APPROVAL"
        artifact["approved_by"] = None
        artifact["approval_timestamp"] = None
    
    # Store artifacts in Redis pending approval
    redis.setex(
        f"artifacts:{workflow_id}",
        86400,  # 24-hour TTL on pending artifacts
        json.dumps(artifacts)
    )
    
    # Notify UI that artifacts are ready for review
    redis.publish(f"workflow:{workflow_id}:events", json.dumps({
        "event": "ARTIFACTS_READY",
        "artifact_count": len(artifacts),
        "workflow_id": workflow_id
    }))
    
    # Wait for user approval (handled by frontend + approval endpoint)
    # Backend receives approval via POST /api/workflows/{id}/approve

async def handle_approval(workflow_id: str, approved_artifact_ids: list, user_id: str):
    # Stamp each approved artifact
    artifacts = json.loads(redis.get(f"artifacts:{workflow_id}"))
    for artifact in artifacts:
        if artifact["id"] in approved_artifact_ids:
            artifact["status"] = "APPROVED"
            artifact["approved_by"] = user_id
            artifact["approval_timestamp"] = datetime.utcnow().isoformat()
    
    # Write to audit stream (immutable)
    redis.xadd("audit_log", {
        "workflow_id": workflow_id,
        "event": "ARTIFACTS_APPROVED",
        "user_id": user_id,
        "artifact_ids": json.dumps(approved_artifact_ids),
        "timestamp": datetime.utcnow().isoformat()
    })
    
    # Mark workflow complete
    redis.hset(f"workflow:{workflow_id}", "status", "COMPLETE")
    
    # Update renewal command center
    redis.zrem("renewals_by_deadline", vendor_id)  # Remove from urgent queue
```
